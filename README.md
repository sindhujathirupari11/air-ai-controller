# ✋ Air AI Gesture Controller

Control your computer using hand gestures in real-time using computer vision.

---

## 🚀 Features

* 🖱️ Mouse control using index finger
* 👆 Click using pinch gesture (thumb + index)
* 🎨 Air drawing mode
* ⚡ Smooth cursor movement
* 🤖 Real-time hand tracking using MediaPipe

---

## 🧠 Tech Stack

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* NumPy

---

## 🎮 How to Use

| Gesture             | Action              |
| ------------------- | ------------------- |
| Index finger        | Move mouse          |
| Thumb + Index close | Click               |
| Press `D`           | Toggle drawing mode |
| Press `Q`           | Exit                |

---

## ⚙️ Setup

```bash
git clone https://github.com/YOUR_USERNAME/air-ai-controller.git
cd air-ai-controller

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
python app.py
```
